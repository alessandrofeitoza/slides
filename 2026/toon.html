Token ≠ Palavra

  • Pontuação vira token
  • Aspas viram token
  • Estrutura vira token

TOON

Text-Oriented Object Notation

  • Estrutura mínima
  • Sem ruído sintático
  • Linear
  • Foco em semântica

JSON vs TOON

JSON


{
  "title": "OO",
  "summary": "Paradigma...",
  "keywords": ["classe","objeto"]
}

TOON


title: OO
summary: Paradigma...
keywords: classe | objeto

Menos tokens. Mesma informação.

   ↓
Context Optimizer
   ↓
LLM Client
   ↓
TOON Parser
  • Separação de responsabilidades
  • Redução de contexto desnecessário
  • Parsing simples
