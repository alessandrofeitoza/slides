Código Ruim

foreach ($users as $user) {
    echo $user->orders()->count();
}

Problema

  • N+1 Query
  • Múltiplas consultas
  • Baixa performance

Código Melhorado

$users = User::with('orders')->get();

foreach ($users as $user) {
    echo $user->orders->count();
}

Redução massiva de queries.

Estudos de Caso

  • Fila travada em produção
  • Redis congestionado
  • CPU em 100%
  • Memory leak em Node.js
  • Deadlock em Laravel
  • Timeout em APIs externas
