Aggregates

Consistência e proteção do domínio

O problema


$item->setQuantidade(-10);
$pedido->setTotal(0);

Objetos expostos permitem estados inválidos.

O que é um Aggregate?

Conjunto de objetos tratados como uma única unidade de consistência.

  • Agrupa Entities e Value Objects
  • Protege invariantes
  • Controla modificações

Exemplo Melhor

  • Mantém consistência

O que é um Domain Event?

Registro de algo importante que ocorreu no domínio.

Eventos descrevem fatos.

Exemplo


class PedidoPago
{
    public function __construct(
        public string $pedidoId
    ) {}
}

Sem Eventos
