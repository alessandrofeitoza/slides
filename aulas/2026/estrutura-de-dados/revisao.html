MODULARIZAÇÃO

Procedimento


function welcome() {
    document.body = "Bem vinde";
}

Procedimento

Apenas executa a ação.

  • Nao retorna nada
  • Não precisa de nada para ser executado

SubRotina


function welcome(nome) {
    return `Bem vinde ${nome}`;
}

SubRotina

Executa uma ação mediante um input e retorna um output.

  • retorna algo
  • precisa de alog para ser executado

ESCOPO DE VARIÁVEIS

Global


var cidade = "Fortaleza";

function welcome(nome) {
    return `Bem vinde ${nome}, de ${cidade}`;
}

Global

Definida no inicio do código e pode ser usada dentro e fora de subrotinas/functions

Local


var cidade = "Fortaleza";

function welcome(nome) {
    let cidade = "Caucaia";
    return `Bem vinde ${nome}, de ${cidade}`;
}

welcome("Chiquim");

console.log(cidade);

Local

Definida dentro de uma subrotina e só pode ser usada dentro dela

Referencia


var cidade = "Fortaleza";

function welcome(nome, &cidade) {
    cidade = "Caucaia";
    return `Bem vinde ${nome}, de ${cidade}`;
}

welcome("Chiquim", cidade);
console.log(cidade);

Referencia

Uma variavel global só pode ser alterada em escopo local se for passada por referencia (&)
