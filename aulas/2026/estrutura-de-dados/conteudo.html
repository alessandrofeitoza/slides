4. Estruturas de Repetição

Como processar muitos dados?


// FOR Clássico (Controle total)
for (let i = 0; i < 5; i++) { ... }

// FOR...OF (Iterar valores de um array)
for (let fruta of frutas) { ... }

// WHILE (Enquanto uma condição for real)
while (saldo > 0) { ... }

Conceito: Iterar

Do latim iterare: "Repetir", "Fazer de novo".

Na computação: É o ato de percorrer uma coleção de dados, um por um, para processá-los.

Imagine um carteiro passando em cada casa de uma rua. A rua é o Array, e o ato de entregar a carta é a Iteração.

LIFO vs FIFO

LIFO

Last In, First Out

"O último a entrar é o primeiro a sair."

Imagine uma Pilha de Pratos: você coloca um em cima do outro. Qual você tira primeiro para não quebrar tudo?

LIFO na Prática (JS)

O Array se comporta como Pilha com push e pop.


const pilha = [];

pilha.push("Página A");
pilha.push("Página B"); // Último a entrar

console.log(pilha); // ["Página A", "Página B"]

const saindo = pilha.pop(); // Tira o último!
console.log(saindo); // "Página B"

FIFO

First In, First Out

"O primeiro a entrar é o primeiro a sair."

A justiça impera: Imagine uma Fila de Banco. Quem chegou primeiro, deve ser atendido primeiro.

Usamos push para entrar e shift para sair pela frente.

O conceito de Callback

"Eu te ligo de volta!"

É uma função passada como argumento para outra função.

A função principal decide quando e como chamar a função que você enviou.

Callback: Exemplo Real


// 1. A função que define o "O QUE" fazer
const enviarEmail = (nome) => {
    console.log(`📧 E-mail enviado para ${nome}`);
};

// 2. A função que define "QUANDO" fazer
const salvarNoBanco = (nome, acaoFinal) => {
    console.log(`💾 Salvando ${nome} no banco...`);
    acaoFinal(nome); // Aqui o callback é executado!
};

salvarNoBanco("Feitoza", enviarEmail);
    

const welcome = (nome) => `Bem vinde ${nome}`;

function login (email, senha) {

    console.log(
        welcome("seu zé")
    );
}

Métodos de "Modificação"

Esses alteram o array original.


let frutas = ["🍎", "🍌"];

frutas.push("🍇");     // Adiciona no fim -> ["🍎", "🍌", "🍇"]

frutas.pop();          // Remove do fim -> ["🍎", "🍌"]

frutas.unshift("🍓");  // Adiciona no início -> ["🍓", "🍎", "🍌"]

frutas.shift();        // Remove do início -> ["🍎", "🍌"]

Performance: O "Pulo do Gato"

Rápido

push / pop

Você só mexe na última gaveta. Ninguém mais se move.

Lento

unshift / shift

Você mexe na primeira gaveta. Todos os outros itens precisam "pular" uma casa.

Isso é a base do Big O que veremos depois!

Encontrando Dados


const convidados = ["Ana", "Beto", "Caio", "Beto"];

// Retorna a primeira posição onde encontrou
console.log(convidados.indexOf("Beto")); // 1

// Retorna true ou false
console.log(convidados.includes("Caio")); // true

// Encontra o primeiro que satisfaz uma condição (callback!)
const busca = convidados.find(nome => nome.length > 3); // "Beto"

Transformação e Filtro

Não mexem no original, criam um novo array.


const nums = [1, 2, 3, 4];

// MAP: Transforma todo mundo
const dobro = nums.map(n => n * 2); // [2, 4, 6, 8]

// FILTER: Seleciona alguns
const pares = nums.filter(n => n % 2 === 0); // [2, 4]

// SLICE: Fatia uma parte (do índice 1 ao 3 - sem incluir o 3)
const parte = nums.slice(1, 3); // [2, 3]

O "Canivete Suíço": Reduce

Reduz o array inteiro a um único valor.


const carrinho = [10.50, 20.00, 5.00];

// acc = acumulador | n = valor atual
const total = carrinho.reduce((acc, n) => {
    return acc + n;
}, 0); // 0 é o valor inicial do acumulador

console.log(total); // 35.50

String ↔️ Array


// SPLIT: Transforma String em Array
const dataStr = "2026-03-30";
const partes = dataStr.split("-"); // ["2026", "03", "30"]

// JOIN: Transforma Array em String
const dataBr = partes.reverse().join("/"); // "30/03/2026"

💡 Reverse também é mutável! Cuidado.

6. High Order Functions

Manipulação moderna de dados.


// forEach: Apenas percorre
nums.forEach(n => console.log(n));

// Map: Transforma e gera novo array
const dobro = nums.map(n => n * 2);

// Filter: Filtra elementos
const pares = nums.filter(n => n % 2 === 0);

// Reduce: Reduz a um único valor (ex: soma)
const total = nums.reduce((acc, n) => acc + n, 0);
