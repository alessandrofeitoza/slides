Pilhas e Filas

  • Pilha: LIFO → Last In, First Out
  • Fila: FIFO → First In, First Out
  • Pilha trabalha com topo
  • Fila trabalha com início e fim
exemplos do mundo real: - Pilha de pratos - Fila de banco

Questão — Pilha

Uma estrutura de dados é um modo particular de armazenamento e organização de dados em um computador.

Selecione a alternativa que mostra o princípio de funcionamento de uma pilha em estrutura de dados.

  • A) LIFO
  • B) FIFO
  • C) FEFO
  • D) LEFO
  • E) FILA
Resposta: A) LIFO

Questão — FIFO

A sigla FIFO refere-se a estruturas de dados do tipo fila.

Como é o funcionamento em uma FIFO?

  • A) O primeiro objeto inserido é o último a ser removido
  • B) O primeiro objeto inserido é também o primeiro a ser removido
  • C) O último objeto inserido é o primeiro a ser removido
  • D) O programador define a ordem
  • E) FIFO e LIFO possuem as mesmas características
Resposta: B) Primeiro a entrar é o primeiro a sair.

Questão — Pilha

Uma estrutura de dados do tipo Pilha segue qual princípio de organização?

  • A) FIFO (First-In, First-Out)
  • B) Random Access
  • C) LIFO (Last-In, First-Out)
  • D) Ordenação Alfabética Obrigatória
  • E) Priority Queue
Resposta: C) LIFO

Questão — Fila de Impressão

O primeiro documento enviado para uma impressora deve ser o primeiro a ser impresso.

Esse comportamento caracteriza qual estrutura?

  • A) Pilha
  • B) Vetor
  • C) Fila
  • D) Árvore
  • E) Nenhuma das alternativas
Resposta: C) Fila (FIFO)

Questão — Pilha x Fila

Se inserirmos os elementos A, B e C:

  • Na primeira estrutura, o primeiro elemento removido é A
  • Na segunda estrutura, o primeiro elemento removido é C

Quais são essas estruturas respectivamente?

  • A) Pilha e Fila
  • B) Fila e Deque
  • C) Fila e Pilha
  • D) Lista Circular e Pilha
  • E) Fila e Árvore Binária
Resposta: C) Fila e Pilha

Questão — Histórico do Navegador

Qual estrutura é ideal para implementar o botão “Voltar” de um navegador?

  • A) Fila
  • B) Matriz
  • C) Pilha
  • D) Vetor
  • E) Nenhuma das alternativas
Resposta: C) Pilha

Questão — Bubble Sort

No Bubble Sort, qual é o procedimento básico?

  • A) Divide o array ao meio
  • B) Compara elementos adjacentes e realiza trocas
  • C) Escolhe elemento aleatório
  • D) Remove os elementos da memória
  • E) Usa árvore AVL
Resposta: B) Comparar adjacentes e trocar

Questão — Complexidade

Avalie as afirmativas:

  • 1 - Quick Sort executa em O(n log n)
  • 2 - Selection Sort executa em O(n)
  • 3 - Algoritmos abaixo de O(n log n) não usam comparação
  • A) Somente a 1 está correta
  • B) Somente 1 e 2
  • C) Somente 2 e 3
  • D) Somente 3
  • E) Todas
Resposta: A) Apenas a 1 está correta

Questão — Busca Binária

Para que a Busca Binária funcione corretamente:

  • A) Os dados devem ser apenas positivos
  • B) Os dados devem estar ordenados
  • C) A lista deve ter potência de 2
  • D) Deve usar pilha
  • E) Deve estar na nuvem
Resposta: B) Dados ordenados

Questão — Dividir e Conquistar

A técnica Dividir e Conquistar consiste em:

  • A) Ignorar metade dos dados
  • B) Somar todos os valores
  • C) Dividir o problema em subproblemas menores
  • D) Utilizar apenas um loop
  • E) Duplicar o problema
Resposta: C) Dividir em subproblemas menores

Questão — Nó Raiz

Em uma árvore, o nó do topo é chamado de:

  • A) Interior
  • B) Terminal
  • C) Raiz
  • D) Exterior
  • E) Filho
Resposta: C) Raiz

Questão — Algoritmos Avançados

Sobre algoritmos avançados em Estruturas de Dados:

  • A) Hash serve apenas para ordenação
  • B) Árvores binárias facilitam busca, inserção e remoção
  • C) Grafos representam apenas estruturas lineares
  • D) Algoritmos avançados sempre são mais lentos
  • E) Busca binária funciona em dados desordenados
Resposta: B) Árvores binárias facilitam operações
