Implementando a Collection

class UserCollection {
  constructor() {
    this.users = [];
  }

  add(user) {
    this.users.push(user);
  }

  createIterator() {
    return new UserIterator(this.users);
  }
}
A Collection gerencia os dados. O Iterator gerencia a navegação.

Implementando o Iterator

class UserIterator {
  constructor(users) {
    this.users = users;
    this.index = 0;
  }

  hasNext() {
    return this.index < this.users.length;
  }

  next() {
    return this.users[this.index++];
  }
}
  • Encapsulamento da navegação
  • Controle interno
  • Desacoplamento

Utilizando o Iterator

const users = new UserCollection();

users.add("Alessandro");
users.add("Sara");
users.add("João");

const iterator = users.createIterator();

while (iterator.hasNext()) {
  console.log(iterator.next());
}

Symbol.iterator Protocolos Iteráveis nativos

Explodindo a Cabeça da Turma

class Range {
  constructor(start, end) {
    this.start = start;
    this.end = end;
  }

  [Symbol.iterator]() {
    let current = this.start;
    let end = this.end;

    return {
      next() {
        if (current <= end) {
          return {
            value: current++,
            done: false
          };
        }

        return { done: true };
      }
    };
  }
}

Usando o Range

const range = new Range(1, 5);

for (const number of range) {
  console.log(number);
}
