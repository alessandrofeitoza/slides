Paradigma Orientado a Objetos

  • Código representa o mundo real
  • Dados + comportamento juntos
  • Responsabilidades claras

Por que focar em OO?

  • Domínio complexo
  • Sistema cresce com o tempo
  • Facilita manutenção
  • Comunica melhor com o negócio

Projeto: Escola de Natação

  • Gestão manual
  • Inadimplência
  • Pouca visibilidade financeira
  • Donos sem tempo

Código – Aluno

	
class Aluno
{
    public function __construct(
        private Plano $plano
    ) {}

    public function pagarMensalidade(): Pagamento
    {
        return new Pagamento($this->plano->valor());
    }
}

Código – Plano e Pagamento

	
class Plano
{
    public function __construct(
        private float $valor
    ) {}

    public function valor(): float
    {
        return $this->valor;
    }
}

class Pagamento
{
    public function __construct(
        private float $valor
    ) {}
}

Anti-OO (Exemplo Errado)

	
class Sistema
{
    public function pagarAluno($aluno, $plano)
    {
        if ($plano['valor'] > 0) {
            // faz tudo aqui
        }
    }
}
