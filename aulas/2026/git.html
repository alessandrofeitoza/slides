$_ git status

Identificar quais arquivos foram adicionados/excluidos/modificados





$_ git diff

Visualizar as modificações


- $t = 'valor';
+ $T = 'valor';

$_ git add file01.txt file02.txt

Prepara os 2 arquivos para serem enviados





$_ git add diretorio/

Prepara todos os arquivos daquele diretório para serem enviados





$_ git add .

Prepara TUDO que foi modificado para ser enviado





$_ git commit -m "comentario"

"Etiqueta" as mudanças, tente responder a pergunta "o que essa alteração faz?"


Cria tal coisa...
Valida o form...
Configura a conexão...

$_ git pull origin main

Traz as mudanças que houveram no repositório





$_ git push origin main

Envia as suas mudanças (commits) para o repositório





Em pleno 2025: "Ei, vou dar um push, dá o pull aí"

GIT do jeito certo

$_ git branch nome-da-branch

Lista, cria e organiza ramificações do projeto


feature/login
bugfix/issue-3
hotfix/producao


$_ git checkout feature/login

Troca de branch sem alterar histórico





$_ git checkout -b feature/login

Cria e já troca de branch





$_ git commit --amend

Adiciona as alterações no mesmo commit anterior





$_ git remote update

Atualiza referências remotas sem alterar sua branch local


Ver o que mudou no remoto
Sem risco de conflito

$_ git merge

Combina o histórico de duas branches


Une trabalho isolado
Preserva a linha do tempo

$_ git rebase

Move commits para outra base, criando um histórico linear


Histórico mais limpo
Menos commits de merge
