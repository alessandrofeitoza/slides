A biblioteca de fazer slides em HTML

Criado por Hakim El Hattab e contribuidores

Como funciona?

O reveal.js permite criar apresentações interativas e bonitas usando HTML.

Primeiro passo

Criar o arquivo HTML

	
	<html>
		<head>
			<link rel="stylesheet" href="dist/reveal.css" />
			<link rel="stylesheet" href="dist/theme/white.css" />
		</head>
		<body>
			<div class="reveal">
				<div class="slides">
					<section>Slide 1</section>
					<section>Slide 2</section>
				</div>
			</div>
			<script src="dist/reveal.js">
			<script>
				Reveal.initialize();
			</script>
		</body>
	</html>
Como "instalar"?

Slides Verticais

Slides podem ser aninhados dentro uns dos outros.

Use a tecla Espaço para navegar por todos os slides.

Nível Subterrâneo 1

Slides aninhados são úteis para adicionar mais detalhes sob um slide horizontal de alto nível.

Código Bonito

Sintaxe colorida cortesia de highlight.js.

		
		
		<section>
			<pre data-id="code-animation">
				<code class="hljs php" data-line-numbers="|1-2|8-10">
					<?php
						for ($i = 0; $i <= 10; $i++) {
							echo "Carro Pancadão Hey <br>";
						}
					?>
				</code>
			</pre>
		</section>
